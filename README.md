# 🧠 Operational Risk & DORA Analytics Dashboard

An interactive risk analytics dashboard built using Python and Streamlit to monitor operational and compliance risks using real-time market data and intelligent risk modeling.

## 🚀 Features

- 📊 Real-time market data integration (Yahoo Finance)
- ⚙️ Key Risk Indicators (KRI): ICT incidents, downtime, control effectiveness
- 🧠 Composite risk scoring (Low / Medium / High classification)
- 📰 Live financial news ingestion with sentiment analysis
- 🚨 Automated risk alerts (email-based escalation)
- 📋 DORA-aligned compliance risk register with dynamic scoring
- 📈 Interactive dashboard with KPI cards and visual analytics

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- yFinance

## 🎯 Use Case

Designed for operational risk, non-financial risk (NFR), and compliance teams to monitor risk exposure, identify early warning signals, and support data-driven decision-making.
